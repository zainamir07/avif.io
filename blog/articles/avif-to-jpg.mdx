## Introduction

The AV1 image file format is an excellent image file format because of its superior compression properties. They allow for images with smaller file size with excellent visual quality. Unfortunately, it is not supported by many software and hardware solutions. Consequently, it is not surprising that people are looking for tools to convert AVIF to JPG. Several tools are available to help you convert an AVIF file to the more commonly used JPG format.

### ImageMagick

<Link href="imagemagick.org" text="Link" />

ImageMagick is a free utility for creating, editing, composing and converting images. It supports over 200 formats. In addition to resizing, flipping, mirroring, rotating, distorting, shearing, and transforming images, ImageMagick can adjust image colors, apply various special effects, and draw text, lines, polygons, ellipses, and BÃ©zier curves.

With version 7.0.25, AVIF image conversion and compression have been natively supported. With the "-define" command-line option, you can specify additional settings for the output JPG file.

<Syntax language="xml">{`magick -quality 75 input.avif output.jpg`}</Syntax>

### Sharp

<Link href="sharp.pixelplumbing.com/" text="Link" />

This Node.js high-performance module converts large images in standard formats into smaller, web-friendly JPEG, PNG, WebP, GIF, and AVIF images of varying sizes. Due to the use of libvips, resizing an image is typically 4x-5x faster than using the quickest ImageMagick and GraphicsMagick settings.

All color spaces, embedded ICC profiles, and alpha transparency channels are handled correctly. Resampling by Lanczos ensures that quality is not sacrificed for speed. In addition to image resizing, other options include rotation, extraction, compositing, and gamma correction.

Modern systems running Node.js >= 12.13.0 on macOS, Windows, and Linux do not require additional install or runtime dependencies. For more information, please visit: https://sharp.pixelplumbing.com

<Syntax language="xml">
  {`sharp("bear.avif")
.jpg()
.resize({ width: size })
.toFile("bear.jpg") }`}
</Syntax>

### Squoosh CLI

<Link href="github.com/GoogleChromeLabs/squoosh/tree/dev/cli" text="Link" />

We had already discussed Squoosh, but we will discuss the CLI this time. Squoosh v2 was announced at Chrome DevSummit 2020, along with a command-line interface to bring all the codecs of Squoosh to the command line using Node.js and WebAssembly.

By using the CLI's, you can let it choose the codec parameters for you and compress entire folders with a single command. With npx, the Squoosh CLI can be used directly from the command line without installation. If you desire, you may also install the Squoosh CLI. You can find more information on their official Github.

To parallelize the processing of images, the Squoosh CLI uses a worker pool. In this way, Squoosh can encode multiple photos simultaneously with the same codec. Squoosh CLI is not the fastest image compression tool available, and it does not intend to be. Nonetheless, it is fast enough to compress many images simultaneously.

<Syntax language="xml">
  {`squoosh-cli --jpg '{quality: 100}' input.avif`}
</Syntax>

The Squoosh CLI also includes an experimental auto optimizer that compresses an image to the maximum, aiming to achieve a specific Butteraugli target value. As Butteraugli target value increases, the output can introduce more artifacts. Using "auto" as the configuration object will enable you to take advantage of the auto optimizer.

<Syntax language="xml">{`npx @squoosh/cli --wp2 auto test.png`}</Syntax>

## What is AVIF anyways?

From keyframes of the video format AV1, the Alliance for Open Media developed the new image format AVIF in collaboration with Google, Cisco, and Xiph.org. Netflix has acknowledged that the AVIF image format provides superior image quality in file size compression compared to JPEG, PNG, and WebP. While JPEG has been around for decades, it is time for a new, more advanced alternative. This makes AVIF an exciting development with significant adoption potential.

AVIF can be used in a variety of ways. As a result, it replaces PNG, JPEG, and WebP in images, including photos and graphics. As AVIF files are much smaller than JPEGs, businesses should use them since they can compress several megabytes into bytes without sacrificing any quality. Furthermore, this reduces data consumption and speeds up your website's loading speed, making information more easily accessible and facilitating more conversions.

AVIF support is already present in Google Chrome, Mozilla Firefox, Opera, Android Browser, and Samsung Internet. Microsoft Edge and Safari are still to follow.

AVIF is not alone. JPEG XL is a raster graphics file format that offers lossless compression and lossy compression. It is designed to outperform existing raster formats and thereby replace them universally.
